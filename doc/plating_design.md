## 最终功能描述（无消耗数值版）

### 一、移动与机动性
| 功能名 | 功能描述                                   |
|--------|----------------------------------------|
| 灵魂出窍 | 激活后视角脱离身体自由飞行（可穿过方块），本体静止且不无敌；再次按右键结束。 |
| 高跳 | 将玩家抛向空中，并在和跳跃提升一样给予一定时间的摔落伤害减免         |
| 冲刺 | 使用物品时向前突进                              |
| 无下坠 | 射出的箭矢不会下坠。                             |
| 土狼时间增强 | 从方块边缘离开后的2秒内仍可触发跳跃。                    |

### 二、战斗与攻击
| 功能名 | 功能描述                                      |
|--------|-------------------------------------------|
| 格挡 | 手持武器右键格挡，减免正面伤害50%并降低击退。                  |
| 伤害 | 直接增加基础攻击伤害。                               |
| 暴击 | 在除了跳砍之外的情况下也有概率触发暴击                       |
| 爆伤 | 进一步提高暴击伤害倍率。                              |
| 头颅抢夺 | 击杀生物时有概率掉落其头颅。                            |
| 追踪 | 射出的箭矢/弹射物轻微追踪附近生物。                        |
| 直钩无下坠 | 鱼钩无重力直线飞行；钩中实体后收竿，强制将其拉回至使用者位置。           |
| 破坏直接进包 | 破坏方块时掉落物直接进入背包。                           |
| 击杀直接进包 | 击杀生物后掉落物直接进入背包。                           |
| 破坏直接吃废石 | 破坏**石头、圆石、深板岩**等普通石质方块时，方块不产生掉落物，而是转化为以太。 |

### 三、资源与采集
| 功能名 | 功能描述 |
|--------|----------|
| 矿透 | 按住某键时，视野内的矿石高亮透视显示；松开关闭。 |
| 时运 | 提高矿石等方块的掉落数量倍率。 |
| 抢夺 | 提高生物掉落物品数量。 |

### 四、以太流系统
| 功能名 | 功能描述                                                                                |
|--------|-------------------------------------------------------------------------------------|
| 以太流穿行 | 玩家附生以太流能量束，沿鼠标方向高速移动（可穿透实体，不可穿透方块）。                                                 |
| 以太流伤害 | 以太流形态下触碰实体造成伤害。                                                                     |
| 发射以太流破坏方块 | 发射以太流射线，破坏前方方块（有距离限制）。                                                              |
| 抗黑暗 | 免疫失明/黑暗效果，低光照下保持可见视野。                                                               |
| 伦理镀层（剑/斧） | 攻击被动生物（牛、羊等）不致死，直接掉落对应产品，生物不会受惊吓逃跑（无冷却限制）。攻击中立生物（铁傀儡、狼等）直接取消仇恨（不再攻击玩家）。攻击敌对生物无特殊效果。 |
| 抗音爆 | 免疫音爆类伤害（如循声守卫声波）。                                                                   |
| 静音跑步 | 跑步、跳跃、落地不产生脚步声，不惊动循声守卫。                                                             |

### 五、生存与环境适应
| 功能名 | 功能描述 |
|--------|----------|
| 自带水中（陆地激流） | 陆地上也能使用三叉戟的“激流”冲刺效果。 |
| 抗黑暗 | 免疫失明/黑暗效果，低光照下保持可见视野。 |
| 抗音爆 | 免疫音爆类伤害（如循声守卫声波）。 |
| 仁慈虚空 | 掉入虚空时传送至Y=319（或255），并清除摔落伤害。 |
| 水上行走 | 可在水、熔岩、粉雪表面行走而不下沉。 |

### 六、特殊效果与伦理机制
| 功能名 | 功能描述 |
|--------|----------|
| 伦理镀层（剑/斧） | 攻击被动生物（牛、羊等）不致死，直接掉落对应产品，生物受惊吓逃跑（无冷却限制）。攻击中立生物（铁傀儡、狼等）直接取消仇恨（不再攻击玩家）。攻击敌对生物无特殊效果。 |
| 原地不动变箱子 | 站立不动3秒后伪装成箱子（视觉+碰撞箱），移动或受击解除。 |
| 动态光源 | 手持发光物品时自身成为移动光源。 |

### 七、其他/辅助
| 功能名 | 功能描述 |
|--------|----------|
| 耐摔 | 抵消衰落伤害（免疫摔落伤害）。 |
| 创造飞行 | 生存模式下自由飞行（类似创造模式飞行）。 |
