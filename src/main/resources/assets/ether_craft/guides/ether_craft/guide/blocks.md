# 方块

## 以太矿石、深层以太矿石、下界以太矿石

- 普通镐破坏掉落失活以太。任意镀层镐破坏会掉落以太水晶和失活以太

## 以太玻璃

模组特供无边框玻璃，但是一堆特性。

<FloatingImage src="assets/glass.png" align="right" />

### 生产

- 以太流经过普通玻璃时可能将其变为以太玻璃
- 以太流打到玻璃掉落物会将其变为以太玻璃

### 特性

- 以太玻璃不会碎裂，且可以用扳手快速拆除
- 以太流在以太玻璃内穿行时会减少一个固定量的飞行消耗，以及取消随时间增大的飞行时长消耗
- 按住alt后玩家可以透过以太玻璃对其后面的物品进行正常交互（但走不过去）

这样可以搭建更美观的封闭产线，且不影响操作机器。

## 装饰方块

- **失活以太块**：装饰方块
- **平滑失活以太块**：装饰方块
