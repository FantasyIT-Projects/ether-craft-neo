# 以太适应节点

以太适应节点是模组的核心机器，负责**生产以太**以及执行各种自动化任务。

设定上来说，以太与正常物质不大相容，容易消散，所以生产、运输、使用以太之类的工作都要放到一个"适应节点"里进行。以太适应节点制造出来时没有任何功能，是一个空的方块，玩家必须像改装机器一样为它加入各式各样的**插件**，它才能开始运行。

<FloatingImage src="assets/node.png" align="right" />

## 灯条

以太节点内有一个"灯条"，鼠标悬浮可以看见机器内的 E 值。随着 E 提升，灯条逐渐由深蓝变白。

## 插件系统

插件分为两类：

- **功能插件**：放在 GUI 左侧，决定节点的核心功能（如生产以太或消耗以太执行任务）
- **升级插件**：放在 GUI 右侧，包括特性插件和升级插件

功能槽无论如何都只有**一个**，而升级槽的数量随以太适应节点的等级提升而增加。

部分对机器的改造会体现在以太节点的外观上，欢迎玩家的发现。
