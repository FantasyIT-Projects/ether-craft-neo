# 功能插件

功能插件决定以太适应节点的核心功能，每个节点只能安装**一个**功能插件。功能插件大致分为"生产以太"和"消耗以太"两类。

## 生产以太

### 熔炉

消耗燃料产出以太。最基础的生产方式。

### 高炉

消耗燃料产出以太，燃烧时间不变，产出略微提高。

### 切石机

消耗可再生废石产出以太。可使用圆石或玄武岩，玄武岩产出速率为圆石的两倍，燃烧时间不变。

### 砂轮

消耗附魔装备或附魔书产出以太。产出 E 与魔咒条数、等级相关。

### 以太水晶

自动、缓慢、无消耗地产生以太。E >= 1000 时停止工作，可用于一些需求量不大但不搭产线的地方。

### 反物质芯片

生存模式最终的生产手段。销毁任何通入的物品，根据物品数量产出以太。

### 创造以太

测试用。放在功能槽位时可以调整每 tick 产生多少 E。

## 消耗以太

### 真空管道

消耗以太吸取指定位置的掉落物。

### 骨粉

消耗以太催熟附近的作物。

### 幽匿催发体

消耗以太催熟附近作物及水晶（紫水晶母岩等）。

### 附魔台

消耗大量以太对物品进行自动化附魔，可提前选择附魔等级。

### 汇流芯片

消耗以太对装备进行镀层。关于镀层详见镀层章节。
