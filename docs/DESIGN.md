# 以太工艺 (Ether Craft) — 设计文档

## 一、以太的本质

以太是构成万物的基本元素（古典以太定义），近代才被发现和利用。玩家是第一批探索者，没有远古文明遗产。

**视觉风格**：混合风格——机器外观是科技感的（金属、机械结构、冷色调），以太能量效果是神秘发光的（蓝紫色光效、粒子效果）。

---

## 二、以太的形态链

以太矿石（主世界/下界矿脉，簇大小8）
  → 熔炼 → 以太物品（1个=100能量，可堆叠64）
  → 一组(64个)丢出静止约5秒 → 失活以太（惰性残渣，基础建材）
  → 注入机器 → 能量形态（存储在机器内部，通过EtherContainer接口管理）
  → 高度浓缩 → 以太流（可编程能量束实体，随时间自然衰减）

- **以太矿石**：主世界（y=-64~32，每区块4簇）和下界（y=0~128，每区块2簇）均有生成
- **以太物品**：熔炼矿石获得，1个=100以太能量
- **失活以太**：一组以太物品丢出静止后转化，是早期基础建材
- **以太锭**：通过加工中心复杂合成（金/铁/铜锭 + 加热芯片 + 多次注能芯片 + 汇流芯片）
- **以太块**：9个以太锭合成，用于存储和高级合成
- **以太玻璃**：特殊玻璃，以太流穿过时减少衰减
- **以太水晶**：失活以太经9次注能芯片加工而成，用于自动补给插件

### 3.2 以太加工中心 (Ether Process Factory)

基于网格的自动化加工机器，四级：

| 等级 | 网格 | 输入行 | 加工格 | 输出行 |
|------|------|--------|--------|--------|
| LV1 | 3x3 | 3 | 9 | 3 |
| LV2 | 5x5 | 5 | 25 | 5 |
| LV3 | 7x7 | 7 | 49 | 7 |
| LV4 | 9x9 | 9 | 81 | 9 |

**配方结构**：树状加工流程

输入物品（左列） -> 加工步骤（内部网格，芯片定义） -> 输出物品（右列）

每个配方由三部分组成：
1. input[]  — 输入物品定义（物品ID + 下一步节点ID）
2. process[] — 加工步骤定义（节点ID + 所需芯片 + 下一步节点ID）
3. output   — 输出物品定义（节点ID + 产出物品列表）

**加工路径**：芯片放入内部网格后，系统自动检测从输入到输出的树状路径。多个输入流可在加工节点汇聚。

**芯片机制**：
- 每个芯片有 maxEther（容量）、etherDecay（衰减周期）、etherRequire（启动需求）、etherConsume（加工消耗）、maxDurability（耐久度）
- 芯片通过衰减环机制周期性消耗以太
- 以太从机器注入芯片，芯片间通过路径传播
- 耐久度耗尽后芯片销毁

**压力加成 (Pressure Bonus)**：当机器以太总量超过所有芯片容量之和时，加工速度提升（对数增长）。

**泄漏 (Leak)**：不匹配的路径（多输入多输出交叉）产生泄漏，消耗额外以太。

**支持多输出**：一个配方可产出多个物品（如奶酪配方同时产出奶酪x4+桶）。

### 3.3 以太流发射器 (Ether Stream Emitter)

独立方块，可旋转方向。以太>1000时自动发射以太流，携带物品栏中的物品。可通过节点插件增强功能。

## 四、以太流系统（E2解锁）

以太流是携带以太能量的飞行实体。

**基本特性**：
- 自然衰减：随时间消耗以太，速度越快衰减越快
- 以太玻璃：穿过时减少衰减，用于长距离传输
- 最大寿命：约10分钟（12000 ticks）
- 大小与以太量相关：size = 0.03 * log10(ether)
- 可穿透特定方块（通过 ether_stream_pass_through 标签）

**能力系统**（通过节点升级插件附加到以太流）：

| 能力 | 功能 |
|------|------|
| 物品存储 | 拾取地面物品，碰撞容器/玩家时自动存入 |
| 方块破坏 | 用工具破坏方块，支持作物收获留种复种 |
| 实体伤害 | 用武器攻击实体（通过假玩家实现） |
| 携带实体 | 拾取并携带实体/玩家飞行，潜行脱离 |
| 作物催熟 | 加速作物生长（可升级为加速所有方块） |
| 镀层 | 为物品施加镀层配方 |
| 反弹 | 碰撞后反向弹回（不消耗以太） |
| 物品显示 | 显示携带物品的图标 |
| 成本缩减 | 降低以太消耗（每级减半） |
| 文字粒子 | 从书籍中发射文字粒子 |

## 五、镀层系统（E3解锁）

为盔甲/物品附加特殊能力。

**镀层流程**：
1. 以太流携带材料 + 目标物品 -> 碰撞触发镀层配方匹配（精确覆盖匹配算法）
2. 匹配成功 -> 物品进入镀层进行中状态（约5秒，显示蓝色粒子）
3. 以太流持续为物品充能（以太量决定效果强度）
4. 镀层完成 -> 获得永久效果

**效果公式**：effect = clamp(ether, a1, a2) 线性映射到 a3~a4
- a1: 最低以太需求
- a2: 最高以太上限
- a3: 最低效果值
- a4: 最高效果值

**充能方式**：以太流碰撞镀层物品/盔甲架/书架/玩家装备时自动分配充能。以太耗尽后镀层清除。

## 六、处理芯片系统

芯片通过数据包定义（data/ether_craft/ether_process_chip/），每个芯片有：
- maxEther：最大以太容量
- etherDecay：衰减周期（tick）
- etherRequire：启动加工所需的以太
- etherConsume：每次加工消耗的以太
- maxDurability：最大耐久度（0=无限）
- behavior：可选特殊行为

## 七、完整进度线

### E1 — 发现以太

挖以太矿 -> 熔炼 -> 以太物品
-> 一组(64个)丢出静止约5秒 -> 失活以太
-> 失活以太合成：
  - 扳手（4个失活以太）
  - 以太适配节点 LV1（8个失活以太）
  - 以太加工中心 LV1（8个失活以太 + 木箱）
  - 隔板芯片（4个失活以太 -> 2个）
  - 熔炉芯片（失活以太 + 熔炉 + 打火石）
  - 冲压芯片（失活以太 + 铁块 + 台阶）
  - 切削芯片（失活以太 + 刀刃）
-> 工厂 LV1 加工：
  - 铁锭 -> 刀刃（加热 + 冲压x2）
  - 金锭 -> 金螺丝（切削x4）
  - 钻石 -> 钻石针（冲压x5 + 切削x2）
  - 失活以太 -> 以太皿（冲压 + 切削）
  - 失活以太 -> 以太水晶（注能x9）

### E2 — 基础加工与以太流

-> 高级芯片合成：
  - 雕刻芯片（失活以太 + 钻石针 + 黑曜石）
  - 成型芯片（失活以太 + 活塞 + 以太皿）
-> 工厂加工：
  - 汇流芯片（以太玻璃 + 失活以太，多步加工）
  - 注能芯片（附魔台 + 真空管道，多步加工）
  - 真空管道（以太玻璃 + 失活以太，复杂多步加工）
-> 以太流发射器（独立方块）
-> 以太适配节点 LV2（失活以太 + 以太皿 + 以太玻璃）
-> 以太加工中心 LV2（LV1升级：金螺丝 + 失活以太）
-> 以太玻璃
-> 以太流升级插件开始可用

### E3 — 以太锭与镀层

-> 以太锭合成（工厂加工）：
  - 金锭/铁锭/铜锭 + 加热芯片 + 多次注能芯片 + 汇流芯片
  - 双倍产出配方（双通道并行加工）
-> 以太块（9个以太锭）
-> 高热芯片（以太锭 + 烈焰棒 + 加热芯片，工厂加工）

-> 以太适配节点 LV3（以太块，工厂加工，19步复杂流程）
-> 镀层系统（以太流携带材料碰撞物品触发）

### E4 — 终局

-> 顶级芯片（工厂加工）：
  - 焕生芯片（嘎枝之心 + 龙蛋 + 以太锭，30步复杂流程）
