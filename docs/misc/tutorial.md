# Ether Craft 模组教程

---

## 条目 1：欢迎

欢迎使用 Ether Craft :)

Ether Craft 是一个以"以太"能量为核心的科技模组。你可以开采以太矿石、搭建自动化加工产线、利用插件系统扩展机器功能，甚至给装备镀上以太涂层获得特殊能力。

翻开下一页，我们从一个最简单的操作开始——获取你的第一份以太。

## 条目 5：获取以太

以太是本模组的核心资源，几乎所有机器和功能都依赖它。

### 开采以太矿石

以太矿石生成于主世界地下，共有三种变体：

- **以太矿石** — 生成于石头层
- **深层以太矿石** — 生成于深板岩层
- **下界以太矿石** — 生成于下界

用铁镐或更高级的镐开采即可获得以太物品。

[截图：以太矿石在地下生成的样子]

### 烧炼以太

将开采到的以太物品放入熔炉烧炼，得到**以太锭**。以太锭是制作机器的基本材料。

[截图：熔炉中烧炼以太]

### 以太的两种形态

- **以太** — 原始形态，放入机器后转化为能量值（默认每份 = 100 以太值）
- **失活以太** — 掉落物形式的以太在 100 tick（5 秒）后会自动转化为失活以太，不再能被机器直接使用

> **提示**：尽快将掉落的以太捡起或导入机器，否则它会失活！

## 条目 10：以太适配节点

以太适配节点是本模组的**核心机器**。它存储以太能量，并通过安装插件来执行各种功能。

### 等级

以太适配节点有 3 个等级：

| 等级 | 特性插件槽 |
|------|-----------|
| LV1 | 2 个 |
| LV2 | 4 个 |
| LV3 | 6 个 |

等级越高，能安装的特性插件越多。

### 放置与打开

手持以太适配节点右键放置。右键点击已放置的节点即可打开 GUI。

[截图：以太适配节点 GUI 主界面]

### GUI 界面

GUI 分为几个区域：

- **左侧**：功能插件槽（1 个）——决定节点做什么
- **右侧**：特性/升级插件槽（2-6 个，取决于等级）——增强或修改功能
- **底部**：物品存储区（最多 27 格）——存放输入和输出物品
- **顶部**：以太能量条——显示当前以太值 / 最大以太值

### 放入以太

将以太物品放入存储区，节点会自动将其转化为以太能量值。默认最大存储为 6400 以太值。

> **提示**：节点默认没有任何功能！你需要先安装一个功能插件，它才会开始工作。详见[插件系统]条目。

## 条目 15：插件系统

插件是以太适配节点的灵魂。没有插件，节点只是一个以太存储罐。

### 两种插件

- **功能插件**（黄色槽位）——决定节点"做什么"。每个节点只能安装 1 个功能插件。
- **特性/升级插件**（蓝色槽位）——修改或增强功能。数量取决于节点等级。

### 安装插件

手持插件物品，**副手持扳手**，右键点击以太适配节点即可安装。

[截图：安装插件时的操作示意]

### 可用功能插件

| 插件 | 功能 |
|------|------|
| **熔炉发电机** | 燃烧燃料产生以太 |
| **高炉发电机** | 燃烧燃料产生以太（效率更高） |
| **磁铁** | 自动吸取附近掉落物 |
| **容器交互** | 与相邻容器交换物品 |
| **投掷器** | 将物品投掷到指定方向 |
| **节点合成** | 使用以太合成物品（类似工作台） |
| **以太转换器** | 将物品直接转化为以太 |
| **装备发电机** | 消耗装备产生以太 |
| **生长加速器** | 加速周围作物生长 |
| **附魔器** | 使用以太进行附魔 |
| **以太流发射器** | 向指定方向发射以太流 |

### 可用特性/升级插件

| 插件 | 功能 |
|------|------|
| **以太存储升级** | 增加最大以太值（倍率 2.5×） |
| **自动补给** | 以太低于阈值时自动补充 |
| **红石信号** | 根据以太值输出红石信号 |
| **物品化以太** | 允许从节点中提取以太物品 |

> **提示**：每个功能插件都有独立的配置界面。安装插件后，GUI 顶部会出现对应的选项卡，点击即可切换。

## 条目 20：以太加工厂

以太加工厂是本模组的**高级加工机器**。它使用工艺芯片组成的树形配方来加工物品。

### 等级与网格

以太加工厂有 4 个等级，网格大小逐级增加：

| 等级 | 网格 | 输入/输出行数 |
|------|------|--------------|
| LV1 | 3×3 | 3 行 |
| LV2 | 5×5 | 5 行 |
| LV3 | 7×7 | 7 行 |
| LV4 | 9×9 | 9 行 |

### GUI 布局

[截图：以太加工厂 GUI]

- **左侧列**：输入槽——放入原材料
- **中间网格**：芯片槽——放置工艺芯片
- **右侧列**：输出槽——成品输出
- **顶部**：以太能量条
- **底部**：过滤器槽——限制每行可接受的输入物品

### 工作原理

1. 在输入槽放入原材料
2. 在芯片网格中按配方排列工艺芯片
3. 工厂自动消耗以太，推进加工进度
4. 进度达到 100 后，输出成品

### 压力加成

当工厂的以太值超过当前配方所需时，多余的以太会转化为**压力加成**，加速加工速度。以太越多，加工越快。

### 以太泄漏

部分配方会导致以太随时间泄漏。注意监控以太值！

## 条目 25：工艺芯片与配方

工艺芯片是以太加工厂的核心耗材。每种芯片有不同的属性。

### 芯片属性

每个工艺芯片具有以下属性：

- **最大以太容量** — 芯片能存储多少以太
- **以太衰减** — 每 tick 自然损耗的以太
- **以太需求** — 开始工作所需的最低以太
- **以太消耗** — 每次加工消耗的以太
- **最大耐久** — 芯片能使用多少次

### 配方答案物品

配方答案物品用于快速配置加工厂的芯片布局：

- **5×5 配方答案**
- **7×7 配方答案**
- **9×9 配方答案**

手持配方答案物品右键，打开**配方获取 GUI**。选择目标产物后，物品会存储对应的芯片网格布局。将其放入加工厂即可自动摆放芯片。

[截图：配方获取 GUI]

### 直接输入芯片

`direct_input` 是一种特殊芯片，用于将输入物品直接传递到下一级加工节点，不消耗以太。

### 芯片快速放置

在加工厂 GUI 中，**手持扳手**拖过芯片网格可以快速放置同种芯片。滚动鼠标滚轮切换芯片类型。

> **提示**：芯片有耐久限制，用完后会损坏消失。记得备好替换芯片！

## 条目 30：以太流发射器

以太流发射器用于在机器之间**传输以太和物品**。

### 放置与朝向

放置时发射器会面向你。手持**扳手**右键可以旋转朝向。

### 工作原理

当发射器内以太值超过 1000 时，它会自动向朝向方向发射一道**以太流**。以太流会携带发射器中的所有以太和输入物品，飞向目标。

[截图：以太流在空中飞行的样子]

### 槽位

- **9 个输入槽** — 放入要传输的物品
- **9 个内部槽** — 内部缓存
- **无输出槽** — 物品通过以太流传输

> **注意**：以太流有最大存活时间（默认 1200 tick = 60 秒），且飞行过程中会持续消耗以太。确保目标在可达范围内！

## 条目 35：以太玻璃

以太玻璃是一种特殊的透明方块，用于**控制以太流的穿透行为**。

### 基本属性

- 像普通玻璃一样透明
- 硬度较低，可被任何工具破坏

### 穿透交互

**按住 Alt 键**时，以太玻璃会变为可穿透状态（无碰撞箱）。这让你可以穿过玻璃与以太流交互，而以太流在穿过玻璃时消耗会减少（默认减少 20 以太值）。

### 用途

- 构建以太流管道走廊
- 在以太流路径上创建观察窗口
- 减少长距离以太流的能量损耗

## 条目 45：镀层系统

镀层系统让你可以给装备（盔甲、武器、工具）附加特殊能力。

### 工作原理

1. 制作镀层物品
2. 将镀层物品和装备一起放入加工厂或适配节点
3. 消耗以太完成镀层
4. 装备获得对应的特殊效果

### 可用镀层效果

| 镀层 | 效果 | 适用装备 |
|------|------|---------|
| **冲刺** | 消耗以太向前冲刺 | 护腿 |
| **高跳** | 消耗以太跳得更高 | 靴子 |
| **灵魂投影** | 脱离身体自由观察周围 | 头盔 |
| **无重力** | 射出的箭不受重力影响 | 弓/弩 |
| **土狼时间** | 离开边缘后仍可跳跃 | 靴子 |
| **伪装** | 静止不动时隐身并回复以太 | 盔甲 |
| **格挡** | 正面伤害减半 | 盾牌/武器 |
| **暴击** | 攻击必定暴击 | 武器 |
| **暴击伤害** | 暴击伤害增加 | 武器 |
| **猎头** | 击杀生物有概率掉落头颅 | 武器 |
| **追踪箭** | 箭矢自动追踪目标 | 弓/弩 |
| **破坏入包** | 破坏的方块直接进入背包 | 工具 |
| **击杀入包** | 击杀生物掉落直接进入背包 | 武器 |
| **吸石** | 破坏石头类方块获得以太 | 工具 |
| **以太流冲刺** | 沿以太流快速移动 | 护腿 |
| **以太流伤害** | 以太流可造成伤害 | 武器 |
| **以太流破坏** | 以太流可破坏方块 | 工具 |
| **抗黑暗** | 免疫黑暗/失明效果 | 头盔 |
| **道德** | 特殊道德行为 | 盔甲 |
| **抗音爆** | 免疫监守者的音爆攻击 | 盔甲 |
| **静步** | 抑制振动事件 | 靴子 |
| **耐久吸收** | 消耗以太代替耐久损耗 | 任意装备 |

> **注意**：所有镀层效果都需要消耗以太才能触发。如果以太不足，效果不会生效！

## 条目 50：升级合成

升级合成是一种特殊的合成方式，可以在合成结果中**保留原物品的 NBT 数据**。

### 使用方式

升级合成配方在普通工作台中使用。配方中标记了"复制来源"的格子——合成结果会继承该格中物品的所有 NBT 标签。

这主要用于：
- 升级已镀层的装备而不丢失镀层效果
- 将附魔从旧工具转移到新工具

> **提示**：如果你不确定某个配方是否支持升级合成，查看 JEI/REI 中的配方说明。

## 条目 55：奶酪与杂项

### 奶酪

*"小巧便携，随时补充你的状态栏！"*

奶酪是一种特殊的食物：
- 食用后**清除所有状态效果**（正面和负面）
- 始终可以食用（即使饱食度已满）
- 回复 2 点饱食度和 1.0 饱和度

[截图：奶酪物品]

### 创造模式以太

`ether_creative` 是创造模式专用物品，放入机器后提供无限以太。仅在创造模式下可用。

### 以太块

装饰/存储方块，由 9 个以太锭合成。也可反向分解。

### 以太矿石

主世界、深层、下界三种变体。用铁镐及以上开采。

## 条目 100：配置文件参考

Ether Craft 的所有数值都可以在配置文件中调整。配置文件位于 `.minecraft/config/ether_craft.toml`。

### 核心以太设置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `ether.convert` | 100 | 每份以太物品转化的以太值 |
| `ether.inactivate_convert_tick` | 100 | 以太掉落物失活所需 tick |

### 适配节点设置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `node.default_max_ether` | 6400 | 节点默认最大以太值 |
| `node.upgrade_slots` | [2,4,6] | 各等级的特性插件槽数 |
| `node.furnace.ether_per_tick` | 25 | 熔炉发电机每 tick 产出 |
| `node.magnet.ether_per_stack` | 100 | 磁铁每吸取一组物品消耗 |
| `node.process.max_progress` | 100 | 节点合成最大进度 |
| `node.growth_accelerator.range` | 1 | 生长加速器扫描范围 |
| `node.enchanter.ether_costs` | [6000,12000,24000] | 附魔器三级消耗 |

### 以太流设置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `ether_stream.max_tick` | 1200 | 以太流最大存活 tick |
| `ether_stream.consumption_factor` | 0.005 | 每 tick 基础消耗比例 |
| `ether_stream.ether_glass_prevent_consume` | 20 | 玻璃减免消耗量 |

### 镀层设置

各镀层效果的冷却时间、以太消耗均可在配置文件中独立调整。详见配置文件中的注释。

> **提示**：修改配置文件后需要重启游戏才能生效。
