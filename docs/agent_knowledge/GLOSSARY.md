## 九、进度阶段 (E1~E4)

| 英文 | 中文 | 解锁内容 |
|------|------|---------|
| E1 — Discovery | E1 发现以太 | 挖矿→熔炼→失活→基础合成→LV1机器 |
| E2 — Basic Processing | E2 基础加工与以太流 | 高级芯片→以太流系统→LV2 |
| E3 — Ether Ingot | E3 以太锭与镀层 | 以太锭→镀层系统→LV3 |
| E4 — Endgame | E4 终局 | 顶级芯片→LV4→三大分支 |
